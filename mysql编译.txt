				LNMP测试环境环境搭建
系统版本：Centos7.6(1810)
				1.安装mysq
安装相关软件包  yum -y install cmake ncurdes ncurses-devel  bison perl-Module-Install.noarch
Mkdir /data/mysql 创建数据存储位置（等同于/var/mysql）
Group mysql 创建组
Useradd -g mysql ; -r -s /sbin/nologing mysql 创建运行用户
编译
 CMAKE -DCMAKE_INSTALL_PREFIX=/usr/local/mysql  \安装目录
-DMYSQL_DATADIR=/data/mysql \  数据存储目录
> -DMYSqL_unix_ADDR=/tmp/mysql.sock \ mysql.sock路径 
> -DWITH_INNOBASE_STORAGE_ENGINE=1  启动innodb引擎
-DEXTRA_CHARSETS=all 启用额外的字符集类型（默认为all）
 -DDEFAULT_CHARSET=utf8  设置默认字符集为utf8
-DDEFAULT_COLLATION=utf8_general_ci 设定默认排序规则（utf8_general_ci快速/utf8_unicode_ci准确
-DWITH_READLINE=1 启用readline库支持
-DWITH_EMBEDDED_SERVER=1 编译嵌入式服务器支持
-DENABLED_LOCAL_INFILE=1 启用本地数据导入支持
-DWITH_DEBUG=0           禁用debug
Make &&  make  instal （内存小可能编译不成功）
改权限
Cd /usr/local/mysql 
Chown  -R root . 
Chown  -R mysql data/
Chown  -R /data/mysql
更改my.cnf
cp support-files/my-default.cnf /etc/my.cnf 2>> $install_log
 sed -i '18c basedir = /usr/local/mysql' /etc/my.cnf
sed -i '19c datadir = /data/mysql'
sed -i '22c socket = /tmp/mysql.sock' /etc/my.cnf
初始化软件  /usr/local/mysql/scripts/mysql_install_db--user=mysql--datadir=/data/mysql/ --basedir=/usr/local/mysql/
安装完执行  mysql_secure_installation
